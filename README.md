## Abstract
本研究では、PLATEAU LOD2の建物データをOpenStreetMapにインポートするためのコンバーターの開発を目指す。従来と異なり、LOD2のデータに対応する点で新規性があり、3D年モデルの取り組みやデジタル地図の充実に貢献するものである。
